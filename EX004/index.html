Por quê existe essa hierarquia?

Essa hierarquia é útil na hora de organizar o texto. Pode ser extremamente útil na hora de montar uma pauta de conteúdo e também de um índice.
Atualmente o HTML 5 é focado no significado dos comandos e não mais na forma. Para isso utilizamos o CSS

Quantos níveis existem?

Os níveis de um título vão de h1 até h6

O número dos níveis não tem haver com o tamanho das fontes e sim com a organização semântica do texto. Para alterar o tamanho das fontes, seu estilo e cores utilizamos o css
