Tamanhos de fontes

Quando escrevemos um texto para uma página web, se não indicarmos nenhum tamanho em específico os navegadores tem por padrão apresentar o texto em 16px.

Quando queremos usar um tamanho diferente o ideal é seguir o que recomenda o W3C. Como temos uma grande variedade de estilos e tamanhos de telas o ideal é adotar uma medida padrão. Os padrões mais utlilizados são o pixel(px) e a porcentagem (em).

Peso e redimensionamento de fontes

Para trabalhar com fontes é importante saber seu peso. O que significa se ela é mais fina ou mais grossa, se é mais escura ou mais clara, além do seu tamanho. Para isso utilizamos as tags font-family, font-weight, font-size e font-style

Todas estas propriedades podem ser sintetizadas na aplicação da tag font seguida dos valores relativos a cada uma delas na mesma ordem descrita acima. Com isso temos o seguinte:

  • Atenção para esses detalhes
  • font-family - Para escolher o tipo de letra;
  • font-weight - peso da fonte, se ela é mais escura, mais grossa ou mais clara;
  • font-size - tamanho da fonte. No exemplo usamos a medida em, mas pode ser em pixels também. Sendo essas duas unidades as mais adequadas para uma página web;
  • font-style - estilo da fonte, pode swer negrito, itálico ou sublinhada;
  • Tudo isso pode ser sintetizado ustilizando a tag font como na figura. Desde que seguindo a ordem acima descrita.
