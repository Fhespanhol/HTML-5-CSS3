Formatação de textos

Negrito / destaque

Para colocar em negrito existem duas maneiras: A primeira é usando a tag b , a segunda é usando a tag strong O resultado visual é o mesmo como visto abaixo:

Contudo no primeiro caso temos apenas a formatação visual e no segundo caso a formatação semântica do texto onde a tag strong informa ao navegador que aquele texto destacado é importante.

Texto em itálico / ênfase

Assim como no negrito temos duas formas de colocar um texto em itálico:
A primeira é usando a tag I e a segunda usando a tag em. Visualmente o resultado é o mesmo como visto nos exemplos abaixo:

Usando a tag em Texto em itálico
Contudo no primeiro caso temos apenas a formatação visual e no segundo caso a formatação semântica do texto onde a tag em informa ao navegador que aquele texto destacado é importante.

Formatação de textos monoespaçados

Um texto monoespaçado é aquele onde todas as letras ocupam o mesmo espaço. Neste texto por exemplo isto não acontece. Cada letra ocupa um espaço. O que acaba fazendo com as letras fiquem truncadas.
Para criar um texto mais elegante visualmente utlizamos a tag code que faz com que o texto fique com todas as letras ocupando o mesmo espaço, ao invés de cada uma de um jeito. Veja o exemplo abaixo:
texto normal
Usando a tag code
Como podemos ver o texto fica com um formato diferente e mais legível.

Código fonte

A tag code também serve para formatação de linhas de código de um programa. Por exemplo, você quer descrever uma instrução em Python pode usar a tag code para diferenciar a fonte do restante do texto, já que ela é perfeita para trabalhar com códigos. Veja no exemplo abaixo:
TRECHO DE CÓDIGO EM PYTHON
a=10
b=20
x=a*b
print('x')
Agora o mesmo trecho utilizando a tag code
a=10
b=20
x=a*b
print('x')

pré formatação

Quando escrevemos textos em html a página gerada tem uma formatação padrão bem simples. Por exemplo, se simplesmente for escrevendo fica tudo na mesma linha. Para mudar de linha é preciso usar a tag br e para mudar de parágrafo a tag p , e não é possível mudar o espaçamento entre uma linha e outra.
Por isso utlizamos a tag pre , a tag pre permite criar um texto respeitando a formatação com que ele foi digitado. Veja o exemplo:
