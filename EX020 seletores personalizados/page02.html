Os divisores fornecem funcionalidades extras aos elementos de uma página web. Com eles podemos criar menus responsivos de maneira simples. Como nas caixas acima, onde ao passar o mouse sobre elas, a cor muda.

Atenção as regras

Um id só pode ser atribuído a um único item, um class pode ser atribuído a qualquer elemento que atenda aquela característica.

Quando descrevemos um id ou class em HTML5 utilizamos estas denominações dentro da tag relacionada ao seletor. Por exemplo: (h1 id="principal")

Ao fazer isso quando declararmos este id na folha de estilo css o título principal adquire as características específicas que determinamos.
