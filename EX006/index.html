Criando listas

Listas ordenadas

  1. Acordar
  2. Levantar
  3. Tomar café
  4. Peidar

Como pudemos ver listas ordenadas são as listas numeradas.
Para criar uma lista numerada utilizamos a tag ol e /ol e dentro dela a tag li sempre antes de cada item da lista. Com isso criamos uma lista ordenada de maneira simples.
Outra vantagem de criar uma lista ordenada é que na hora de incluir ou excluir um item a numeração é atualizada automaticamente.

Listas ordenandas variadas

Vocè não precisa ficar limitado somente a este tipo de lista, para mudar o modelo basta usar a tag type dentro da tag ol . Isto vai permitir que você trabalhe com outros modelos de listas. Vamos a elas:

Listas não ordenadas

São as listas que não necessitam de uma ordem lógica. Basta inserir os elementos. Para inserir este tipo de lista utilizamos a tag ul . Por padrão a lista não ordenada é criada com o círculo preto no início.
Para mudar isto utlizamos a tag type. E os modelos que temos são:

  1. Circle (Bola sem preenchimento);
  2. disc(Bola preenchida em preto);
  3. Square(Quadrado).

Vejamos os exemplos:

Misturando os modelos

É possível também criar listas mescladas, onde temos um modelo não ordenado dentro de um modelo ordenado. Como no exemplo abaixo:

  1. Ir a faculdade
    • Fazer o trabalho
    • Entregar ao professor
    • Jogar papel no chão
    • Colar a porta
  2. Voltar para casa
    • Abrir a porta
    • Fechar a porta
    • Acender a luz
    • Fazer carinho no gato
