Aprendendo a criar uma estrutura de navegação

Para criar uma estrutura de navegação entre páginas é preciso usar internos.
Os links internos permitem que você navegue entre as páginas do site. É o que chamamos de âncoras.

Como inserir um link interno

Para inserir um link interno é preciso usar a tag "a" e dentro dela segurar control + espaço para selecionar a página que será acessada.
