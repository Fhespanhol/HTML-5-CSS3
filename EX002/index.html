Parágrafos e quebra de linhas

Você pode escrever o que quiser dentro das tags de um parágrafo. Isso facilita a sua vida.

Para inserir uma quebra de linha em um lugar específico é preciso usar a tag br

Usando a tag
Como visto ao usar esta tag passamos para a linha de baixo.

Vamos aprender a gerar símbolos especiais usando a tabela que está no PDF do curso. Por exemplo Marca registrada e copyrighting é ® © respectivamente

Vamos aprender a inserir emoji

Para isso vá no Google e pesquise emojipedia. Abra o site que é de onde vamos pegar os códigos dos emoji. Embora copiar e colar diretamente possa funcionar na maioria dos navegadores o ideal é usaro código hexadecimal referente ao emoji. Isso evita que o símbolo não abra.

Para inserir a saudação vulcan 🖖 basta inserir & # x mais o código do emoji que é if596.
